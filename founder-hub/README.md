# Founder Hub

Founder Hub is a private, local-only marketing and analytics workspace for Forgeko.

Run it separately from the public Forgeko product:

```bash
cd founder-hub
npm install
npm run dev
```

Open `http://127.0.0.1:3030`.

Real secrets belong only in `founder-hub/.env.local`. Generated Founder Hub data is stored locally in `founder-hub/.local/founder-hub-snapshot.json`, which is ignored by Git.

Data flow:

- The home page reads the latest private local snapshot from `.local`.
- `Extract latest data` reads public RSS/news feeds, public Reddit JSON, optional Forgeko landing analytics, and then writes one local snapshot.
- RSS news is read from public technology/business feeds; NewsAPI is not required.
- Reddit uses public JSON endpoints only, without OAuth credentials.
- Post drafts are generated by a deterministic local TypeScript engine using Forgeko positioning, source links, news, and Reddit pain points.
- Export downloads a Markdown view of the local snapshot.

Analytics:

- Forgeko analytics is optional and read-only through `FORGEKO_SITE_URL` plus `FOUNDER_HUB_ANALYTICS_TOKEN`.
- Supabase can remain behind the public landing waitlist, but Founder Hub does not use Supabase as its own database.
- Conversion rates show `n/a` when tracked visits/clicks are incomplete or would create misleading percentages.

Founder Hub never posts, comments, sends email, charges users, or modifies external service data.
